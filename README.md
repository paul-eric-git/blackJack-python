BLACKJACK - or at least my Python command-line version of it...

	Fully playable command-line Blackjack game written in Python. Built as a personal project
	to practice concepts as game logic, randomization and clean code structure.

	Features:
		-Full 52-card deck with shuffling
		-Dealer that follows standard Blackjack procedure
		-Real-time score calculation including Ace-handling
		-Dealers 2nd card hidden
		-Win, loss and draw detection
	
	Runs on:
		Python3.x

	How to Play:
		1. Dealer deals you and himself 2 cards, one dealer card is hidden
		2. Choose whether to draw another or not by pressing y/n
		3. If your score exceeds 21, game over!
		4. Once you pass (n) the dealer takes turn and plays until end of match
		5. highest score without going over 21 wins
